Getting Started With KA Lite

KA Lite can run on any Linux or Windows server, with minimal requirements. To get started:

1. Sign up

  • First sign up for an account on this site.
  • The next step will be to create an organization and zone. An organization is an entity responsible for administering multiple different zones and facilities. Organizations can have multiple administrators. A zone is the area in which multiple devices may connect to share content with each other. A facility is the place where the server is located that enables sharing within a zone.

    • 2. Install

  • Next, follow the installation instructions to get KA Lite up and running on your server. All information regarding installation, including running the server and setting up KA Lite can be found in the installation instructions.

    • 3. Register device

  • After installation, you should run the server as shown in the installation instructions. You will be prompted to create an account and register your device on your server.
