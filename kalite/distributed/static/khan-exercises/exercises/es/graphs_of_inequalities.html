Para encontrar la ecuación de una desigualdad lineal primero debes encontrar la ecuación de la recta que forma el límite del conjunto solución. Esta recta se muestra en el gráfico anterior.

Una forma de encontrar la ecuación de esta recta es elegir dos puntos en la línea y encontrar la pendiente y la intersección con el eje y. Estos dos puntos en la recta son (X1,Y1) y (X2,Y2).

Sustituye ambos puntos en la ecuación para la pendiente de la recta.

m = \dfrac{Y2 - negParens(Y1)}{X2 - negParens(X1)} = fractionReduce( Y2 - Y1, X2 - X1 )

Para encontrar b, podemos sustituir cualquiera de los dos puntos en la ecuación cuya pendiente conocemos.

Using the first point (X1, Y1), substitute y = Y1 and x = X1:

Y1 = (fractionReduce( Y2 - Y1, X2 - X1 ))(X1) + b

b = Y1 - fractionReduce( X1 * ( Y2 - Y1 ), X2 - X1 ) = fractionReduce( Y1 * (X2 - X1) - X1 * ( Y2 - Y1 ), X2 - X1 )

La ecuación de la recta es y = ( SLOPE === -1 ? "-" : ( SLOPE === 1 ? "" : fractionReduce( Y2 - Y1, X2 - X1 ))) x + fractionReduce( Y1 * (X2 - X1) - X1 * ( Y2 - Y1 ), X2 - X1 ) (el valor de m es SLOPE).

Ahora que tenemos la ecuación para la línea límite, necesitamos decidir qué signo de desigualdad usar.

Si elegimos un punto sobre la recta, digamos (X1,Y1), podemos ver que los puntos debajo de la recta están sombreados. Estos son los puntos donde x = X1 pero y es menor que Y1 . Por lo que debemos usar un < o signo ≤.

If we pick a point on the line, let's say (X1,Y1), we can see that points above that point are all shaded in. These are the points where x = X1 but y is greater than Y1 . So we should use a > or ≥ sign.

Otra forma de ver esto es evaluar en un punto. El punto mas fácil a evaluar aquí es (0,0):
