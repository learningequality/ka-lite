Las proporciones pueden escribirse de distintas formas que en realidad significan lo mismo.

Puedes expresar una proporción con dos puntos separando los dos números.

NUM:DENOM

Se puede escribirla como una frase como esta.

NUM \text{ TO } DENOM

o puedes expresar una proporción como una fracción.

\dfrac{NUM}{DENOM}=fractionReduce( NUM, DENOM )

Por lo tanto, fractionReduce( NUM, DENOM ) es la proporción de fruit( 0 )s a fruit( 1 )s escrita como una fracción simplificada.
