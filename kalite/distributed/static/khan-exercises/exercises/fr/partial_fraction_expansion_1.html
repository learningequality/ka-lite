D'abord, on factorise le dénominateur pour trouver les dénominateurs des deux fractions simples.

DENOM = (ADENOM)(BDENOM)

Comme le dénominateur est le produit de ces deux facteurs, la fraction peut s'écrire comme la somme de deux fractions dont les dénominateurs respectifs sont ces facteurs.

\dfrac{NUMER}{ (ADENOM)(BDENOM) } = \dfrac{?}{ADENOM} + \dfrac{?}{BDENOM}

Maintenant, on remplace les numérateurs par des polynômes ayant un degré de moins que celui qui est au dénominateur.

Dans notre cas, les deux dénominateurs ont un degré de 1, donc on remplacer les numérateurs par des polynômes de degré 0 ou des constantes. On utilise A et B pour simplifier les notations.

\dfrac{NUMER}{ (ADENOM)(BDENOM) } = \dfrac{A}{ADENOM} + \dfrac{B}{BDENOM}

Maintenant, pour se débarrasser des fractions, on multiplie par le dénominateur commun (ADENOM)(BDENOM).

NUMER = A(BDENOM) + B(ADENOM)

Maintenant, on peut trouver A et B. Un moyen facile de le faire est d'essayer de choisir des valeurs de x qui annulera A ou B puis trouver l'autre.

Essayons d'éliminer B. On voir que si on remplace x par C, le terme en B s'annule et il nous reste :

expr(["+", E * C, F]) = A(expr(["+", C, -D]))

E * C + F = expr(["*", C - D, "A"])

A=A

On peut faire de même pour trouver B, mais on remplace x de préférence par D :

expr(["+", E * D, F]) = B(expr(["+", D, -C]))

E * D + F = expr(["*", D - C, "B"])

B=B

Maintenant, on remplace les valeurs trouvées dans la fraction et on obtient :

\dfrac{NUMER}{DENOM} = \dfrac{A}{ADENOM} + \dfrac{B}{BDENOM}
