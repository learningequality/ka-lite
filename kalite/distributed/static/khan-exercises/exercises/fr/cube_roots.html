\Large{\sqrt[3]{Q} = \text{?}}

N

\sqrt[3]{Q} est le nombre qui, lorsqu'il est multiplié trois fois par lui-même, vaut Q.

Nous cherchons \sqrt[3]{Q}, donc nous allons répartir les facteurs premiers en trois groupes identiques.

Il n'y a que trois facteurs premiers, et nous voulons les répartir en trois groupes, ce qui est donc facile.

Q = PRIMES.join( "\\times " ), donc N^3 = Q.

Remarquez qu'on peut réorganiser les facteurs ainsi:

Q = PRIMES.join( "\\times " ) = \left([ F_N.join( "\\times " ), F_N.join( "\\times " ), F_N.join( "\\times ") ].join( "\\right)\\times\\left(" )\right)

Donc \left(F_N.join( "\\times " )\right)^3 = N^3 = Q.

Donc N^3 = Q.

Donc \sqrt[3]{Q} is N.
