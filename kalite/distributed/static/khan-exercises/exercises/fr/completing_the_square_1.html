x^2 + Bx \color{blue}{ + pow( B / 2, 2 )} = C * -1 \color{blue}{ + pow( B / 2, 2 )}

Nous pouvons maintenant réécrire le membre de gauche comme un nombre au carré.

Le membre de gauche de l'équation est déjà un carré parfait. Le coefficient de notre terme en x est B, la moitié duquel est B / 2, et le mettre au carré nous donne \color{blue}{pow( B / 2, 2 )}, soit notre terme constant.

( x + B / 2 )^2 = C * -1 + pow( B / 2, 2 )
