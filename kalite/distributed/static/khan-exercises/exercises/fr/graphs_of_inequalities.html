Une des manières de trouver l'équation d'une courbe est de choisir deux points de cette courbe et chercher sa pente ainsi que son intersection avec l'axe des y. On choisit les points suivants : (X1,Y1) et (X2,Y2).

On utilise les coordonnées des deux points dans l'équation de la pente d'une courbe.

m = \dfrac{Y2 - negParens(Y1)}{X2 - negParens(X1)} = fractionReduce( Y2 - Y1, X2 - X1 )

Pour trouver b, on peut utiliser l'un des deux points dans l'équation avec la pente trouvée.

En utilisant le premier point (X1, Y1), remplacez par y = Y1 et par x = X1:

Y1 = (fractionReduce( Y2 - Y1, X2 - X1 ))(X1) + b

b = Y1 - fractionReduce( X1 * ( Y2 - Y1 ), X2 - X1 ) = fractionReduce( Y1 * (X2 - X1) - X1 * ( Y2 - Y1 ), X2 - X1 )

L'équation de la courbe est y = ( SLOPE === -1 ? "-" : ( SLOPE === 1 ? "" : fractionReduce( Y2 - Y1, X2 - X1 ))) x + fractionReduce( Y1 * (X2 - X1) - X1 * ( Y2 - Y1 ), X2 - X1 ) (La valeur de m est SLOPE).

Si on choisit un point de la courbe, disons (X1,Y1), on peut voir que les point en dessous sont tous dans la zone grisée. Ces points sont ceux où x = X1 mais y est plus petit que Y1 . Donc on doit utiliser le signe < ou ≤.
