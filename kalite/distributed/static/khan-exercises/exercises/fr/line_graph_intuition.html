La pente d'une courbe est le rapport de variation sur y sur la variation sur x. On peut aussi dire le rapport entre décalage vertical et le décalage horizontal.

Lorsqu'une pente est positive, si on se déplace en augmentant y, x augmente aussi.

Lorsqu'une pente est négative, si on se déplace en augmentant y, x diminue.

Pour une pente nulle, le changement sur y devrait être nulle quelque soit les deux points considérés.

Pour une pente indéfinie, le changement sur x devrait être nul quelque soit les points considérés.
