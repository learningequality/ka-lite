person(1) has NUM fruit( 0 ) for every DENOM fruit( 1 ). Write the ratio of fruit( 0 )s to fruit( 1 )s as a simplified fraction.

NUM / DENOM

Les rapports peuvent être écrits de différentes façons mais qui signifient toutes la même chose.

Vous pouvez exprimer un rapport avec un point virgule qui sépare les deux chiffres.

NUM:DENOM

Vous pouvez écrire ceçi dans une phrase comme celle-ci.

NUM \text{ TO } DENOM

Ou bien, vous pouvez exprimer ce rapport sous la forme d'une fraction.

\dfrac{NUM}{DENOM}=fractionReduce( NUM, DENOM )

Therefore, fractionReduce( NUM, DENOM ) is the ratio of fruit( 0 )s to fruit( 1 )s written as a simplified fraction.
